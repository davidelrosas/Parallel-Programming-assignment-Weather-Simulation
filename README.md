# Weather Simulation (C, OpenMP)

University project developed as part of a parallel programming course.

## Overview

Implemented a weather simulation in C and parallelized computationally intensive sections using OpenMP.

## Features

- Sequential implementation
- OpenMP parallelization
- Performance comparison between implementations

## Technologies

- C
- OpenMP

## Purpose

The goal of the project was to explore parallel programming concepts and evaluate the impact of OpenMP on execution performance.
